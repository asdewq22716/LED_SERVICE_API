	- หน่วยงานมีระดับกลุ่มงานเข้ามาเกี่ยวข้อง ที่ไม่เหมือน TPQI คือวิธีการส่ง
	- ร่างหนังสือตัดทิ้ง
	- หนังสือที่สร้างเห็นของใครของมัน (สารบรรณกลุ่มมองเห็นทั้งหมดในกลุ่มงานตัวเอง) 
	- ออกเลขแยกการรันตามประเภท
	- ส่งหนังสือระหว่างกลุ่ม จากกลุ่มหนึ่งไปยังอีกกลุ่มต่างสำนัก 
	- ถ้าส่งตามลำดับขั้นไม่ต้องผ่านสำนักไปหารองทีเดียวเลย
	- หนังสือค้างส่ง box search ให้ค้นหาคนและเปลี่ยนให้แสดงเป็นชื่อคน
	- หนังสือภายนอกส่งออก เลขรันแยกเป็น 2 แบบ
	- รับหนังสือภายนอก แยกเลขกันรัน ภายนอก,ภายนอก(ลับ)
	- ทุกคนสร้างหนังสือภายนอกได้หมดตามสิทธิ์ที่ set ไว้ สารบรรณกลุ่มไม่เห็นหนังสือภายนอก
	
	*สร้างหนังสือ
	- ปรับหน้าจอสร้างหนังสือให้สอดคล้องกับรูปแบบของเขา
	- ผู้ตรวจเสนอเปลี่ยนเป็นผู้เสนอ
	- ต้องนำเข้าไฟล์อย่างน้อย 1 ไฟล์ 
	- มุมมองหนังสือตัดทิ้ง
	- มุมมองเอกสาร ส่งออกเอกสาร ตัดทิ้ง
